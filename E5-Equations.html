Kinematics

\[v = \frac{\Delta x}{\Delta t}\] \[a = \frac{\Delta v}{\Delta t}\]

\[v_{fx} = v_{ix} + a_x \, t\] \[x_{f} = x_{i} + v_{ix}\,t + \frac12 a_x \, t^2\] \[v_{fx}^2 = v_{ix}^2 + 2a_x (x_f - x_i)\]

\[v_{fy} = v_{iy} + a_y \, t\] \[y_{f} = y_{i} + v_{iy}\,t + \frac12 a_y \, t^2\] \[v_{fy}^2 = v_{iy}^2 + 2a_y (y_f - y_i)\]

Forces

\[\sum \vec{F} = m \vec{a}\] \[F_g = G \frac{m \, M}{r^2}\] \[f = \mu F_N\]

Work

\[W = F \, d \, \cos(\theta)\] \[W_{net} = \Delta E = \Delta KE + \Delta PE\]

\[KE = \tfrac12 m v^2\] \[PE_g = mgh\] \[PE_s = \tfrac12 k x^2\]

Momentum

\[\vec{p} = m\,\vec{v}\]

\[\vec{I} = \vec{F}~\Delta t\]

\[\Delta \vec{p} = 0\] \[m_1 \vec{v}_{1i} + m_2 \vec{v}_{2i} = m_1 \vec{v}_{1f} +m_2 \vec{v}_{2f}\]

\[\tfrac12 m_1 v_{1i}^2 + \tfrac12 m_2 v_{2i}^2 = \tfrac12 m_1 v_{1f}^2 + \tfrac12 m_2 v_{2f}^2\] \[v_{1i} + v_{1f} = v_{2i} + v_{2f}\]

Rotation

\[\omega_f = \omega_i + \alpha \, t\] \[\theta_f = \theta_i + \omega_i \, t + \tfrac12 \alpha \, t^2\] \[\omega_f^2 = \omega_i^2 + 2 \alpha (\theta_f - \theta_i)\]

\[x \rightarrow \theta\] \[v \rightarrow \omega\] \[a \rightarrow \alpha\] \[F \rightarrow \tau\] \[m \rightarrow I\]

\[x=r\theta \qquad v = r \omega \qquad a = r \alpha\] \[a_{c} = v^2/r\] \[F_{c} = m~v^2/r\] \[KE_{rot} = \tfrac12 I \omega^2\]

\[\tau = r~F~\sin(\theta)\] \[\sum \vec{\tau} = I \vec{\alpha}\]

Moment of Inertia

\[I = \sum m r^2\]

\[\text{Solid sphere} = 2/5~M R^2\] \[\text{Hollow sphere} = 2/3~MR^2\] \[\text{Solid cylinder} = 1/2~MR^2\] \[\text{Hollow cylinder} = MR^2\] \[\text{Rod about center}= 1/12~ MR^2\] \[\text{Rod about end}= 1/3~ MR^2\]
